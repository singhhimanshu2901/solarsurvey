# Solar Survey Phase 1 - Company Portal

## Features
- Demo company login
- Company dashboard
- New survey form
- Roof photo/video upload input
- Roof dimensions: strict/relaxed mode
- Direction, shadow risk, panel count/dimensions
- Rule-based detailed report
- PDF download
- Reports saved in browser localStorage

## Run
```bash
npm install
npm run dev
```

Open the localhost URL shown in terminal.
